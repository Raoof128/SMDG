# Contributing

Thank you for considering a contribution! To keep the project healthy and predictable, please follow these guidelines:

1. Fork the repository and create a feature branch.
2. Run `ruff` and `black` for formatting/linting.
3. Add or update tests with `pytest`.
4. Update documentation for any user-facing changes.
5. Submit a pull request describing the motivation and validation steps.

For significant design changes, open an issue first to discuss the proposal.
